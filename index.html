PROMPT 5 • V9
⏰ ~3 MIN
INTERMEDIATE
Disk & Storage Health: SMART + SSD
SMART attributes → SSD wear → predictive failure
▶ Click to expand
  • HealthStatus, wear level, temperature
  • Read/write latency (Green <10ms, Red >20ms)
  • Event ID 153 = predictive failure
  • Estimated remaining useful life
📌 What You'll Need
HTML Report PowerShell Output
💡 Pro Tip
Event ID 153 is the most critical storage indicator. If present, run this prompt immediately — the drive is retrying failed IOs.
PROMPT 6 • V10
⏰ ~3 MIN
ADVANCED
Baseline Comparison: Drift Detection
Compare two captures → detect drift → trend analysis
▶ Click to expand
  • Event trends: increasing or decreasing?
  • Performance drift: memory leak? storage filling?
  • Configuration drift: new apps, driver changes
  • Trajectory: IMPROVING / STABLE / DEGRADING

Baseline CSV/HTML Current CSV/HTML Same Machine
💡 Pro Tip
Schedule baselines after Patch Tuesday. Your "known-good" already includes latest patches, making drift detection more meaningful.

⏰ ~3 MIN
ADVANCED
Profile + Software Conflicts + Startup
Profile bloat → OST health → conflicts → startup impact
▶ Click to expand

Design Principles

The safety-first philosophy behind every prompt.

🛡

Evidence-First

Never claim root cause without report evidence. Separate confirmed from suspected.

⚠

Safe Before Destructive

Built-in Windows tools first. No OS reset, registry edits, or driver removal without evidence + admin approval.

📋

Audit-Ready

Every action documented with timestamps, Event IDs, and clear reasoning. ServiceNow-ready format.

👥

Business Continuity

Minimize disruption. Prioritize productivity and data safety. Clear escalation criteria.
